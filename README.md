# E-XGBoost: Engineered XGBoost with Focal Loss for Cassava Disease Detection

## Overview
This repository contains the implementation of E-XGBoost, an engineered
XGBoost variant with a custom Focal Loss objective function for early
cassava disease detection in Ghanaian smallholder farms.

The model addresses the class imbalance problem inherent in cassava disease
datasets by fabricating a Focal Loss objective into XGBoost training,
alongside SHAP explainability for farmer-facing insights.
